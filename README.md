# mouse-master-
A Gentle Way to Learn: Sensory-Friendly Mouse Skills

Finding a computer game that is safe, educational, and free from overstimulation can be difficult. This game was built to solve that problem. It offers a focused, quiet space for special needs children to learn essential computer mouse skills without the chaos of the modern web.

With no ads and no internet required, you never have to worry about what your child is clicking on. To ensure healthy habits, the game features a built-in 15-minute session timer that gently ends the turn, requiring a parental password to continue. It is simple, safe, and effective.
